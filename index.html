This Bootcamp has as goal to build a credit application (CreditYa), using Clean Architecture, Spring Webflux , Scaffold Bancolombia, and AWS as cloud provider
Additionally as a personal challenge implement some patterns microservices like Event sourcing, CQRS, and Outbox.
Despite had finished bootcamp project, the main idea is continuing to expand this project adding new features, UI, and patterns microservices.

üìÇ Services

Auth Service
Service for handling users and roles with authentication and authorization using JWT and Spring Security.

Github Repo

User Service
Handles customer data: get by document, create, validate existence. Requires authentication. Built with Scaffold Bancolombia + Webflux.

Github Repo

Loan Service
Manages credits using Event Sourcing and CQRS with MongoDB. Includes Outbox pattern to push messages via AWS SQS.

Github Repo

Notification Lambda
Handles the notifications sending an email using SES service provided by aws service, after being resolved, loan-service pushes an event using SQS for being managed by notification lambda sending an email with the resolution of its loans

Github Repo

Self Decision Lambda
Manages the capacity Credit, when a customer submitted a new credit of type self-decision, checks its salary base and active debts.

Github Repo
