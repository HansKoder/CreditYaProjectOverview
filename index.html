The goal of this Bootcamp is to build a credit application (CreditYa), using Clean Architecture, Spring Webflux, Bancolombia Scaffold, and AWS as the cloud provider.
Additionally, as a personal challenge, I decided to implement microservice patterns such as Event Sourcing, CQRS, and Outbox.
Despite having finished the bootcamp project, the main idea is to continue expanding it by adding new features, a user interface, and more microservice patterns.

Auth Service
Service for handling users and roles with authentication and authorization using JWT and Spring Security.

Github Repo

User Service
Handles customer data: get by document, create, validate existence. Requires authentication. Built with Scaffold Bancolombia + Webflux.

Github Repo

Loan Service
Manages credits using Event Sourcing and CQRS with MongoDB. Includes Outbox pattern to push messages via AWS SQS.

Github Repo

Notification Lambda
The Notification Lambda handles customer notifications.
After a loan application is reviewed by an Advisor (ROLE), the Loan Service pushes a new event to SQS. The Notification Lambda listens to this event and sends an email through AWS SES with the loan resolution.

Github Repo

Self Decision Lambda
The Self-Decision Lambda manages debt capacity.
